Domain dan contoh Domain adalah nama yang dipilih sebagai sebagai identitas server atau computer agar bisa lebih mudah mengingatnya.
Contohnya : facebook.com Smkn1cianjur.sch.id

www (world wide web) suatu ruang informasi yang dipakai oleh pengenal global yang disebut dengan pengidentifikasi sumber seragam untuk mengenal pastis umber daya berguna.
Www : kumpulan sumber daya internet (seperti FTP,telnet,usenet),tekxt hyperlink,file audio,dan Video,dan situs jarak jauh yang dapat diakses dan dicari oleh browser berdasarkan standar Seperti HTTP dan TCP/IP.

domain name : nama unik yang diberikan untuk mengidentifikasi nama server computer seperti server web atau server di jaringan computer atau internet.

URL (uniform resource locator) locator sumber seragam adalah rangkaian karakter menurut suatu format standar tertentu yang digunkan untuk menunjukan alamat suatu sumber seperti dokum dan gambar diinternet.

URI (uniform resource identifier) pengidentifikasi sumber seragam adalah sebuah untaian karakter yang digunakan untuk mengidentifikasi nama,sumber,atau layanan diinternet.

http (hypertext transfer protocol) transfer protocol yang dbisa diimplementasikan kesalah satu protocol internet. (belum menggunkan sertifikat SSL).

.

https (hypertext transfer protocol source) :eksentsi dari hypertext transfer protocol protocol komunikasi internet yang melindungi integritas dan kerahasiaan data antara computer pengguna dan situs.
